# Gemini AI Chatbot

A simple AI chatbot built using Google Gemini API and Streamlit.

## Features
- Conversational AI
- Gemini 2.5 Flash model
- Chat history memory
- Interactive UI with Streamlit
- API key input from UI

## Structure:

gemini-chatbot/
│
├── app.py
├── chatbot_gemini.py
├── requirements.txt
└── README.md

## Tech Stack
Python
Google Gemini API
Streamlit

## Run Locally

pip install -r requirements.txt
streamlit run app.py
